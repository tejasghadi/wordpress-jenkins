== Description ==
Osaka is a Clean, minimal Designed stunning WordPress Theme. Osaka is fully Compatible with Gutenberg Editor and Blocks. Dedicated to Gutenberg. You can use Osaka for Personal Blog/Website, Professional Portfolio, news, magazine, Business Websites, Travel, Biograhpy, Corporate Websites and much more possiblities. Special Features are - Hight Quality Code, Eyecatching crafted Design and built in Gutenberg Support. After using this theme you can easily say that this is best Theme for Gutenberg. Osaka is focused on SEO practices as well, optimized codes for modern SEO factors. If you need any assitance, we're well known for outstanding Supports: https://prowptheme.com/support/


== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the "osaka-light.zip" file. Click Install Now.
3. Click Activate to use your new theme right away.
